# Custom Nginx Docker Image Lab

This project demonstrates how to build and run a custom Nginx Docker image.

## Objective

The goal of this lab is to understand the basic Docker workflow:

1. Create a simple HTML page.
2. Write a Dockerfile.
3. Build a custom Docker image.
4. Run a container from the image.
5. Expose the container to the local machine using port mapping.
6. Test the web server using curl and a browser.

## Tools Used

- Docker
- Nginx
- macOS Terminal
- VS Code
- curl
